Stylist-Lite (Flask + OpenCV)
Учебный проект по дисциплине «Компьютерное зрение»: интерактивный веб-инструмент для локального редактирования цвета объектов (одежды, аксессуаров) на фотографиях с сохранением реалистичных текстур.

Возможности:
- высокоточное выделение объектов любой формы с помощью инструмента «Лассо».
- изменение цвета (HEX), яркости и прозрачности наложения в режиме реального времени.
- функция «Запекания» (Bake) позволяет фиксировать изменения и переходить к перекраске следующего элемента на том же фото.

Стек:
- HTML5 / CSS3 / JavaScript 
- Python / Flask
- OpenCV / NumPy
